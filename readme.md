# BetterStreamer
Eine Webvideokonferenzplattform mit dem Ziel, bessere Videoübertragungsqualität als z.B. Discord zu haben.
Erzielt wurde dies durch Modifizierung von SDP-Paketen. Die Bitratenbegrenzung konnte so nahezu ausgehebelt und Datenraten von bis zu 18000 kBit/s erreicht werden.
